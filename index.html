Linux x86-64 Calling Convention

System Call Convention: System calls use syscall instruction with the syscall number in rax and arguments in specific registers.

System Call Arguments

Argument Register
Syscall Number rax
Argument 1 rdi
Argument 2 rsi
Argument 3 rdx
Argument 4 r10
Argument 5 r8
Argument 6 r9
Return Value rax

Function Call Convention (C ABI)

Argument Register
Argument 1 rdi
Argument 2 rsi
Argument 3 rdx
Argument 4 rcx
Argument 5 r8
Argument 6 r9
Additional args Stack (right to left)
Return Value rax
